Sobre o Projeto

Conheça mais sobre o Dicionário de Dados DER-SP

Objetivo do Projeto

O Dicionário de Dados DER-SP é uma iniciativa para promover a transparência e facilitar o reuso dos dados abertos do Departamento de Estradas de Rodagem do Estado de São Paulo.

Missão

Tornar os dados públicos do DER-SP mais acessíveis, compreensíveis e reutilizáveis através de documentação padronizada e ferramentas interativas.

Visão

Ser referência em transparência de dados públicos rodoviários, facilitando pesquisas, análises e desenvolvimento de soluções baseadas em dados.

Funcionalidades

Consulta Interativa

Interface moderna para busca e filtros avançados nos metadados dos conjuntos de dados.

Publicação Assistida

Sistema automatizado para análise de dados e geração de dicionários com mínima intervenção manual.

Exportação Múltipla

Exportação dos dicionários em diversos formatos: CSV, Excel, PDF e JSON.

Design Responsivo

Interface adaptativa que funciona perfeitamente em desktops, tablets e smartphones.

IA Integrada

Algoritmos inteligentes para inferência automática de tipos de dados e metadados.

Padrões Abertos

Seguimento de padrões internacionais para metadados e dados abertos governamentais.

Roadmap do Projeto

Fase 1 - Concluída
  • Interface de consulta interativa

  • Exportação básica
  • Design responsivo
Fase 2 - Concluída
  • Sistema de publicação assistida

Como Contribuir

Este é um projeto de código aberto e sua colaboração é bem-vinda! Existem várias formas de contribuir:

Reportar Problemas

Encontrou um bug ou tem uma sugestão? Abra uma issue no GitHub com detalhes do problema ou melhoria.

Contribuir com Código

Faça fork do repositório, implemente suas melhorias e envie um pull request.

Melhorar Dados

Ajude a melhorar a qualidade dos metadados e documentação dos conjuntos de dados.
